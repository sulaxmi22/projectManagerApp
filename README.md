# projectManagerApp

Project Management Application uses Node.js with Express.js with REST API convention (Backend), MVC + Service Architecture and use Mongoose.js + Mongo DB (Database) and Vue.js (Frontend).

Steps to run : Backend Node.js + Express.js Application including the config.json which connects to your Mongo DB the server should start by running: "npm run start" Frontend Vue.js Application the dev server should start by running: "npm run dev"
